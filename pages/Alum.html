Full Stack Application - Alumni Networking Directory

AlumLink

AlumLink is an alumni directory solution designed to help graduates stay connected with their alma mater and each other. It enables users to search for alumni, view their professional profiles, and network efficiently through an intuitive web interface. The platform streamlines alumni engagement by providing a centralized hub for communication and career opportunities.

Technologies

React, Node.js, PostgreSQL, Passport.js
