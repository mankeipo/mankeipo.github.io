Stray-Finder

Stray-Finder is a web application designed to help locate and track stray animals in real time. Users can view stray animal locations on a map, report new sightings, and receive updates through email notifications. The platform aims to connect the community and provide a way to ensure strays are found and cared for efficiently.

Technologies

React, Node.js, Google Maps API, Nodemailer, AWS
