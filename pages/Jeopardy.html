Jeopardy With Friends

Jeopardy With Friends is a real-time multiplayer web application designed to replicate the classic TV game show. Players can compete against each other by answering questions across various categories, with the game board displaying different question values.

Technologies

React, Node.js, Firebase, Socket.IO
